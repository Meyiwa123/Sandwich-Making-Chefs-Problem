Sandwich Making Chefs Problem
README

Files:
src/Kitchen.java - Contains the shared table where the ingredients will be placed, and starts the Agent and Chef threads
src/Agent.java - Create an agent that randomly places two unique ingredients on the table
src/Chef.java - Makes and eats a sandwich once the required ingredients are on the table
Documentation/Class Diagram.jpg - The UML class diagram
Documentation/Sequence Diagram.jpg - The UML sequence diagram

To run:
To run the program invoke the main function in Kitchen.java class, which 
will initializa a Kitchen object and run the threads. 
Using Eclipse - Load the project -> Right click on the project title -> Select "Run as" -> Select "Java Application"

Editing ingredients:
To edit the ingredients, change the elements of the ingredients list
to the ingredient of your choice.
